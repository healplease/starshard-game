### Notes / observations (no action required)
- **The boss is structurally immune to every flush.** It lives in `world.boss`, never in the
  `enemies`/`asteroids`/`ebullets` lists that `_flush` clears ‚Äî so both the free arrival clear (R57) and a
  player bomb (R63) leave its HP untouched with no special-case guard. Clean and deterministic.
- **"Free + silent" arrival is verifiable by the residual charge.** Because the f20 bomb left the smoke
  run at charge 1 and the f40 arrival is free, the run **ends at charge 1** ‚Äî a direct, attributable
  assertion that the system clear cost no charge (AC40), exactly as the level-designer's smoke-coexistence
  note (¬ßV7.7) predicted.
- **The 12-red ring is even by construction, not by luck.** Each of the 3 yellow bullets contributes an
  interleaved quarter (`ring_phase ‚àà {0,30,60}` + `k¬∑90¬∞`), so the union is always the even 30¬∞ set
  regardless of the boss‚Üíplayer aim of the telegraph ‚Äî the QA-binding invariant (AC50) holds for any aim.
- **One run clock = no difficulty discount.** `world.t` advances through ARRIVAL‚ÜíDEFEAT, so the ramp, the
  survival tick, and the next mark never pause during the fight. The interlude **swaps** storm for an
  equally-dangerous arena rather than handing out a breather ‚Äî the pacing basis for AC13 holding.
- **AC13 (run length 1‚Äì3 min) ‚Äî non-blocking caveat carried + extended.** The level-designer reconciled
  the breakpoint by **lowering the gate** (boss 1 @75 s is reliably seen before the ~120 s median death;
  bosses 2+ are the late/expert event), with the fight economy-frozen and clock-advancing so it stays in
  the band. The only mild lengthener is the bounded ~5 s calmer entrance, once for the median run. This
  extends the **existing parked "AC13 long runs"** watch-item (`feature_inventory.md ¬ß4`); it does **not**
  block v7. Levers (cheapest-first: `BOSS_HP`, then the marks) live in `level_spec/v7-bosses.md ¬ßV7.6` ‚Äî
