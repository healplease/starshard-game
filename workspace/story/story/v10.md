## V10.3 ★ `CONTROLS_2` rewrite — `"Esc  Pause"` ⚠ REWRITE

> **Old value** (`config.py` line 345): `"Esc  Pause · hold Q  Quit"`
> **New value**: `"Esc  Pause"`

**Why rewrite:** v8 put `"hold Q  Quit"` inside `CONTROLS_2` because back then the gesture was *not*
active on START — that line was only *previewing* the in-pause quit. In v10 the gesture **is** active on
START and gets its own dedicated, arc-anchored line (§V10.2). Leaving the clause in `CONTROLS_2` would
print **"hold Q  Quit" twice on START** (y≈509 and y≈609, ~100 px apart) — clutter, not clarity. Drop
the duplicate; `CONTROLS_2` returns to teaching exactly one binding: **Esc pauses**.

- The quit gesture is **not lost** — it moves to the dedicated `START_QUIT_HINT` line that the arc sits
  under (which is the correct home now that the gesture is live on this screen).
- `"Esc  Pause"` still satisfies **AC59** (the v8 regression check: `"Esc"` and `"Pause"` both present in
  `CONTROLS_2`). No stale "Esc Quit" anywhere (Esc still never quits — v8 R73).

## V10.5 START prompt (`START_PROMPT`) — KEPT `"Press any key to fly"` (no change)

GDD §V10.5 flags that **Q no longer starts the run** (Q is carved out of "any key starts" and reserved
for hold-to-quit), so `"Press any key to fly"` is now technically true of every key *except* Q.

**Decision: keep `"Press any key to fly"` verbatim** (Designer-blessed in §V10.5; the prompt "can stay").

- Pressing exactly **Q** to *start* is a vanishingly rare intent, and a Q **tap** now simply does nothing
  (it only begins/cancels a hold) — no misleading action fires.
- The new **`START_QUIT_HINT`** line directly below disambiguates Q's role: "any key flies — and *holding
  Q* quits." The two lines read as call-to-action + its one deliberate exception.
- `"Press any key to fly"` is iconic, on-voice v1 arcade copy; rewording it to hedge a 0.0001 % edge
  ("press a key", "press Space/Z to fly") would cost clarity and re-open settled v1 copy for no real gain.

Recorded so nothing is left hunting: the START prompt is **intentionally unchanged**; the quit hint, not
the prompt, teaches Q.
